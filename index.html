Cara setup Firebase:
1. Buka console.firebase.google.com
2. Buat project → Realtime Database → Create database
3. Pilih region asia-southeast1 (Singapore)
4. Rules: set .read: true, .write: true (sementara)
5. Project Settings → Your Apps → Add Web App → Salin config
6. Ganti semua nilai GANTI_* di baris ~10 file ini

🛡️ Firebase Rules
Salin ke Firebase Console → Realtime Database → Rules
{
  "rules": {
    "site": {
      ".read":  true,
      ".write": true
    },
    "admin": {
      ".read":  false,
      ".write": true
    },
    "_ping": {
      ".read":  false,
      ".write": true
    }
  }
}

Keunggulan Kami

Mengapa Tarie Florist?

01
🌸
Bunga Segar Premium

Kami bermitra langsung dengan kebun bunga pilihan. Setiap rangkaian dijamin menggunakan bunga segar berkualitas tinggi yang tahan minimal 3–5 hari setelah pengiriman.

02
🎨
Desain Eksklusif

Florist senior kami yang berpengalaman lebih dari 8 tahun akan merancang papan bunga yang unik, personal, dan mencerminkan karakter acara Anda.

03
⚡
Same-Day Delivery

Pesan sebelum pukul 12 siang, papan bunga Anda tiba hari itu juga. Kami beroperasi 7 hari seminggu termasuk hari libur nasional.

04
💎
Harga Transparan

Tidak ada biaya tersembunyi. Harga yang tertera sudah mencakup rangkaian, papan standing premium, pita, dan kartu ucapan personal.
